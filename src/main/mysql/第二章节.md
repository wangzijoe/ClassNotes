# 2-2 CPU资源和可用内存的大小
##### 计算密集型应用
每当双十一大促发生的时候，对于计算密集型的应用，CPU资源很有可能成为整个系统的瓶颈
##### 热数据
热数据的大小超过可用内存的大小的时候，也会成为系统的瓶颈
##### 网路
大量的数据被查询时候，特别是使用MemCache这样的存储引擎做缓存层的时候，当缓存大量失效的时候，就会产生大量的网络传输，从而影响服务器的性能。（通常的做法，升级i/o子系统）
